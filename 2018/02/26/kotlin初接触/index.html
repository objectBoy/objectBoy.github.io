准备

Android Studio 3.0开始，捆绑支持Kotlin，之前版本需要安装插件

  1. File | Settings | Plugins | Install JetBrains plugin…
  2. 搜索Kotlin安装即可

创建一个新的项目（老项目可以无缝转接，跳过此步骤即可），转换java file to kotlin file

  1. 打开MainActivity.java

在项目中配置Kotlin

  1. 编辑文件，或者选择Tools | Kotin | Configure Kotlin in Project from the main

  2. Sync Now

特色

简化findViewById

引入apply plugin:’kotlin-android-extensions’

可以直接使用Id定位界面元素

【注】 Kotlin会遍历你的xml文件，创建对应的虚拟包给你引用（用Alt+Enter引用），我们使用的控件对象，其实是这个虚拟包里的控件对象。它是kotlin临时创建的，你无法打开它的文件，在编译apk时，Kotlin会自动帮你补充findViewbyId的代码，最终得到的产品其实没变，它只是方便了程序员的书写。

语法

相对于java而言

没有“;”

结尾不需要“;”，什么都不用写，并不是向python转化，感觉单纯的为了方便程序员写代码

“:”冒号的多用途

  1. 定义变量类型

  2. 定义参数类型

  3. 定义函数返回值

  4. 声明继承关系

  5. 使用java类

没有”new”

“var” 定义变量 “val”定义常量

延迟加载

  1. Kotlin支持类型推断，但变量必须赋值，java中的基本类型是有默认值的，可以在声明之处不赋值

  2. 延迟加载val/var,第一次调用时初始化

空指针安全

  1. “?”表示可以为空

    声明允许为空时，不能使用类型推断，必须声明数据类型

  2. “!!”表示不可以为空
