准备工作

  • 下载qshell

    qshell 是利用七牛文档上公开的 API 实现的一个方便开发者测试和使用七牛API服务的命令行工具。该工具设计和开发的主要目的就是帮助开发者快速解决问题。

  • 安装

    若不熟悉go语言，请下载对应平台的二进制文件，放到刚才下载的zip解压文件夹中即可

  • 设置环境变量

设置

  • qshell account <ak> <sk>
  • 设置完毕后，即可以通过相应命令，上传和获取外链表，具体的，请参考官方文档此处不再赘述

脚本

  • 主要用到如下方法
    • qshell listbucket
  • 脚本地址
  • sh qiniuyun.sh <bucketName> <newDomainName>

当我告诉自己要忘记你的时候，我便忘不掉了。所以，先启程，努力的生活，踏遍山河，揽尽山川，不经意间，悄然忘却。我相信有那样一个你，在生气，气我停步不前。一定有那样一个你，在等我，微风中青岚的裙摆，鼓鼓的气嘟嘟的笑脸。

先启程，再忘却



> 云淡天清拂风暖，凡尘俗事本无牵。世间多少离情苦，离散无悔尽是缘。

谈到感情，我能想到可以形容自己的词汇，除了白痴，再无其他了。在感情中，很多时候，我并不知道怎样算是适度，何时又该抽身正果。昨晚夜游后海酒吧街后，睡不着，索性就看了《欢乐颂2》。刚巧不巧的，似乎是命运都安排好的，我盯住了何莹莹，目不转睛的盯住了。当然，这并不是我喜欢这个类型的女孩子，相反的，我很讨厌在情感中失去理智的何莹莹。可是…我竟然在这段无厘头的感情片曲中，看到了自己的影子。不，不是影子，应该是翻版。我讨厌这样的何莹莹，就像突然间憎恶这样的自己。

从出生到现在，我经历了2段恋情。一个相处了很久很久，止于谈婚论嫁的前期，恰巧应了七年之痒的诅咒。另一个相处了几个月，止于热恋的前期，又刚好应了三月之痛的诅咒。每一段恋情，我都倾心于当时的时间了，可是，我的感情，存在的意义似乎就是为了给诅咒正名。经历了一番狂风骤雨后，我发现，原来一切都是理所应当的，不是命运故意玩弄，只因，我太白痴。

> 分手后，我该如何自处

每一次巧合，每一次相遇，又或者每一段事情，开始就意味着要结束的。就像我们从生下来就要面对死亡一样，长长短短终究逃不过真理。也因此，白首到老，显得尤弥珍贵，让人如此羡慕。恋爱，就代表着分手的可能性被无限变大，恋爱的开始，是分手从0到非0的本质衍生。就像双生并蒂莲一样，因为爱，因为恨，所以美丽。那么，分手后我该如何自处呢？

曾经，我就像恋情中的何莹莹，不，也许我就是何莹莹。用一些无关紧要，却像是救命稻草一样的理由，麻痹自己，也努力麻痹周围的人，保持着那一丝联系，稍有风吹躁动，内心便包涛汹涌。就像是，屋外下小雨，屋内下中雨。屋外下中雨，我却无法躲到屋外去避雨。我努力的扮演着感情中的受伤者，装作很无辜的样子。我奋力的将自己的痛苦期无线延长，然后努力让感情中真正的受伤者，在愧疚中忘不掉自己。写到这里，我已经想抽自己一顿了。总之，看清自己正在扮演的角色之后，我如此的讨厌自己，恨不得弃自己而去。对，我要弃自己而去。

> 改变，再不改变，我就要扼杀这个讨厌的角色设定了。

其实，还想文绉绉的写下一些自己看法。不过真的等不了了，看到现在的自己，我就恶心的想吐。昨晚，我想了如下几条初步实施方案。

1. 她已经有男朋友了，甚至领了结婚证。我必须滚蛋。不仅滚出她的生活，也要将她请出我的生活。除了手机号，删除所有常用联系方式，不看朋友圈，不看空间，不打听任何关于她的事情。只保留一种最原始的且不稳定的单向联系方式，成为最熟悉的陌生人。我相信，已经有天使爱她，而我该好好地爱自己。
2. 我单身，你也单身。无论中间经历了什么坎坷，我始终是最了解你的那个人。虽然我们必然不可能走到一起了，但是你需要帮忙的时候，我依旧会站在你身边，我是你的朋友，像亲人一样的朋友。而这一切始于我们都单身了，也应该终于，我们有一方不再孤独了。我相信，我们终将会遇到对的人，代替对方，守护彼此。
3. 作为一个男孩子，别整天扮演一个受伤的小狮子的样子。不是我说你，狮群中有你这样的懦夫，真是一件不光彩的事情。我要回到自己的群体，找到狮子的雄心，然后站到最高的山巅，任寒风凌冽，我自笑傲山河。
4. 过去的就过去了，不用刻意忘掉，但也不能随意或者刻意的提起。我相信，以我的超群记忆力，真正的忘记，是再也不会掀起波澜。

> 先启程，再忘却

我该向前走了，启程去看大好河山。

我该向前奔跑了，启程去踏碎洱海的宁静。

我，要放弃自己了。因为，我要迎接一个全新的自己。虽然我不知道，我会不会讨厌那个将要降临的情感灵魂，但是，我至少讨厌现在的自己不能自拔。我不知道这个过程，会经历一个怎样的“快感”，但是无论是苦是甜，我都迫不及待的等待迎接了。

先启程，管他是否忘却。踏破一座山丘，踏碎一条川流，一步一忆，我会寻找更美好的世界。

先启程，管他是否风雨。遇风一起吼，遇雨一起狂。若要潇洒，便先疯狂。

忘却？有必要吗？

我，可以创造更好的回忆~

酒吧街夜景

好啦，现在到家咯。大家晚安~

既然睡不着，那就随便写两句吧。刚刚好，小年刚来未去。人生就像这样，命运总会安排，在刚好的时间，遇到刚好的人。

此时遇到，刚刚好

既然睡不着，那就随便写两句吧。刚刚好，小年刚来未去。人生就像这样，命运总会安排，在刚好的时间，遇到刚好的人。

最近，我和一个女孩子聊得比较多。或者说，我基本只在和这一个女孩子聊天。女孩聪慧，心思细腻。她就像命中注定的一样，出现在我最迷茫的那段日子。也许是命运的怜悯，女孩的出现，就像阴霾中的男孩看到的第一缕光。透过云层，和煦，温暖。

有这样一段时间，男孩从不和周围的人沟通。甚至，很多的同事都不知道他的存在，只知道，教育事业部招了一个新人，也许是个男孩子吧。男孩从不和大家一起吃饭，他总是一个人躲在楼顶的跑道，去看远方的百望山。山的名字，似乎就是为男孩量身定做的，他自己都不知道看过多少次了。男孩记得，朝阳下山的样子，记得烈日下山的样子，也记得余晖下山的样子。他了解山，甚至超过了解那时的自己。男孩只是看着山，不说话。男孩怕，怕张嘴了，就会忍不住哭出来。

领导找男孩谈话，一次又一次。男孩只是听，不说话。最后从舌尖顶出一句谢谢，谈话就算是结束了。男孩知道他惹大家失望了，只是，除了谢谢，男孩不知道该说什么了。他们，毕竟只是陌生人，即便要撑，也要撑作很坚强的样子。

又是一个午后，又是那个足迹罕至的屋顶，又是那个男孩。一样的天空，天空下一样的百望山。唯一不同的是，男孩拿出手机，打了一个电话，是打给一个女孩的。那是男孩第一次开口，第一次说了那么多话，第一次找到了想要诉说的对象…是的，那日，男孩说了很多，努力装作一切都过去了，无所谓的样子。虽然挂断电话后，男孩依然安静的倒在长椅上，看了百望山好久，但我相信，一切是从那通电话开始过去的……真正的过去了。从拨通电话开始，男孩的时间不再静止在过去，开始流向未知了。
命运就是这样，我们都是它的玩具。而它也像是一个长不大的孩子，调皮，却也知道珍惜。总是在关上一扇门的时候，打开一扇窗。只是，这门似乎关的有点久了……是不是出去玩了，忘记了被锁住的我？门终究是没有打开的，而墙壁却多了一扇窗，凿窗的人是一个女孩，不高，爱吃好吃的。她或许是命运的使者，又或许就是我命运中，不可或缺的一部分。

女孩不大，95年生人，小男孩几岁。而她却像是男孩心里的姐姐（男孩总倔强的认为，心里是住着一个姐姐的，从小，一直陪伴着他，守护他，从未离去），出现在了他最需要的时候……刚刚好。男孩感激女孩，就像感激着少虎的心情一样，用男孩的话说，她对男孩是有恩的，男孩希望能为女孩做些什么。
人非草木，熟能生情。渐渐地，男孩视女孩如妹妹，异父异母的亲妹妹。男孩希望女孩开心，健康，平安。虽然这和男孩对所有的朋友的祝福都一样，却唯独多了一分联系。男孩从来不称呼女孩为妹妹，却喜欢在聊天时，发“摸摸头”的表情。他喜欢这个词，很温暖，却不像情侣，不像长辈，更像兄妹，纯粹的温暖。女孩虽小，却似乎也有很多自己的小秘密，满满的心事藏在那笨拙的妆颜下。女孩不说，男孩便也不问，时光静好。

命运就是这样，

在刚刚好的时间，遇到刚刚好的人。

又不知在何时消弭在人海茫茫… …

浮游一生，怀着感恩的心，珍惜每一个刚刚好出现的人，做刚刚好的自己。

你在看吗？此时遇见，刚刚好。
