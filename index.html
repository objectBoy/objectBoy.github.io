Hexo博客搭建

前些日子用hexo搭建了个博客，在此记录下过程，希望能帮助后来者。

准备

  • Github
  • Node.js
  • Git
  • Hexo

步骤

Github

建立与你用户名对应的仓库，仓库名必须为your_user_name.github.io，例如我的用户名是objectBoy那么我建立的仓库名就是：objectBoy.github.io

安装Git

百度一下下载安装即可

安装Node.js

具体的安装流程我就不细说了，这里有一篇很详细的文章，介绍给大家http://www.cnblogs.com/LonelyShadow/p/4310100.html

安装Hexo

Hexo需要在Node.js和git之后安装，执行命令后等待即可

npm install -g hexo <span <span class="hljs-keyword">class</span>=<span class="hljs-string">"hljs-comment"</span>><span class="hljs-comment">// -g指全局安装</span></span>
`</pre>

<span class="hljs-preprocessor"># Hexo</span>

  1. 在电脑中建立一个名字叫「Hexo」的文件夹（比如我建在了D:\Hexo），然后在此文件夹中右键打开Git Bash。执行下面的命令

    <pre><span class="hljs-title">hexo</span> init</pre>

    等待过后，Hexo会在该目标文件建立所需要文件

  2. 安装node_modules

    <pre>npm <span class="hljs-keyword">install</span></pre>

  3. 此时已经可以运行下看看效果了

    <pre>hexo <span class="hljs-keyword">server</span></pre>

    启动服务后，在浏览器中打开http://localhost:4000/，这时可以看到Hexo已为你生成了一篇blog_HelloWorld

    此时的样子是hexo默认的主题，需要更换主题的话，只需要将主题拷贝到theme文件夹中，然后配置_config.yml即可
    具体的配置方法，请参照官网

  4. 创建新的博文

    <pre>hexo <span class="hljs-keyword">new</span> <span class="hljs-string">"My New Post"</span></pre>

  5. 生成静态网页

    <pre>hexo <span class="hljs-keyword">ge</span><span class="hljs-label">nerate</span></pre>

    将博客绑定到GitHub

  • 配置_config.yml文件

    <pre>`# Deployment

    Docs: http://hexo.io/docs/deployment.html

    deploy:
    type:&lt;/pre&gt; 更改为 &lt;pre&gt;# Deployment

    Docs: http://hexo.io/docs/deployment.html

    deploy:
    type: github
    repository: git@github.com:xxx/xxx.github.io.git
    branch: master`</pre>

  • 开始部署

    <pre><span class="hljs-variable">$ </span>hexo deploy</pre>

    在这里我遇到了个问题，说Deployer not found: github
    解决方法为：

    <pre>npm <span class="hljs-operator"><span class="hljs-keyword">install</span> hexo-deployer-git <span class="hljs-comment">--save</span></span></pre>

    然后将配置文件里的github -> git

    再次执行部署命令即可。

    部署完成后再浏览器中http://xxx.github.io/ 查看效果

    每次部署的步骤

    <pre>hexo clean <span class="hljs-comment">//清除缓存</span> hexo generate <span class="hljs-comment">//生成静态网页</span> hexo deploy <span class="hljs-comment">//部署</span></pre>

    本地调试

  1. 在执行下面的命令后，
    <pre><span class="hljs-variable">$ </span>hexo g <span class="hljs-comment">#生成</span> <span class="hljs-variable">$ </span>hexo s <span class="hljs-comment">#启动本地服务，进行文章预览调试</span></pre>
    浏览器输入http://localhost:4000，查看搭建效果。此后的每次变更_config.yml 文件或者新建文件都可以先用此命令调试，尤其是当你想调试新添加的主题时。

  2. 可以用简化的一条命令

    <pre><span class="hljs-title">hexo</span> s -g</pre>

    命令总结

    常用命令

    <pre>hexo <span class="hljs-keyword">new</span> <span class="hljs-string">"postName"</span> <span class="hljs-preprocessor">#新建文章</span> hexo <span class="hljs-keyword">new</span> page <span class="hljs-string">"pageName"</span> <span class="hljs-preprocessor">#新建页面</span> hexo generate <span class="hljs-preprocessor">#生成静态页面至public目录</span> hexo <span class="hljs-keyword">server</span> <span class="hljs-preprocessor">#开启预览访问端口（默认端口4000，'ctrl + c'关闭server）</span> hexo deploy <span class="hljs-preprocessor">#将.deploy目录部署到GitHub</span> hexo help <span class="hljs-preprocessor"># 查看帮助</span> hexo version <span class="hljs-preprocessor">#查看Hexo的版本</span></pre>

    复合命令

    <pre>hexo deploy -g <span class="hljs-preprocessor">#生成加部署</span> hexo <span class="hljs-keyword">server</span> -g <span class="hljs-preprocessor">#生成加预览</span></pre>

    <pre>`hexo <span class=”hljs-variable”>n =</span>= hexo new
