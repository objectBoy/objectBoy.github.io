七牛云外链批量导出

最近一直在用七牛云作为外链，发一些游记图片。可是图片多了，一个个的去复制外链却也麻烦，所以去看了下七牛的api，写了一个批量导出外链的脚本，下次再写游记导出图片就不是骑牛啦，得开车了，快上车吧~

准备工作

  • 下载qshell

    qshell 是利用七牛文档上公开的 API 实现的一个方便开发者测试和使用七牛API服务的命令行工具。该工具设计和开发的主要目的就是帮助开发者快速解决问题。

  • 安装

    若不熟悉go语言，请下载对应平台的二进制文件，放到刚才下载的zip解压文件夹中即可

  • 设置环境变量

设置

  • qshell account <ak> <sk>
  • 设置完毕后，即可以通过相应命令，上传和获取外链表，具体的，请参考官方文档此处不再赘述

脚本

  • 主要用到如下方法
    • qshell listbucket
  • 脚本地址
  • sh qiniuyun.sh <bucketName> <newDomainName>
