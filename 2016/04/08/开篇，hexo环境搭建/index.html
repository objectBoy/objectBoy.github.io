Hexo博客搭建

前些日子用hexo搭建了个博客，在此记录下过程，希望能帮助后来者。

准备

  • Github
  • Node.js
  • Git
  • Hexo

步骤

Github

建立与你用户名对应的仓库，仓库名必须为your_user_name.github.io，例如我的用户名是objectBoy那么我建立的仓库名就是：objectBoy.github.io

安装Git

百度一下下载安装即可

安装Node.js

具体的安装流程我就不细说了，这里有一篇很详细的文章，介绍给大家http://www.cnblogs.com/LonelyShadow/p/4310100.html

安装Hexo

Hexo需要在Node.js和git之后安装，执行命令后等待即可

  1. 在电脑中建立一个名字叫「Hexo」的文件夹（比如我建在了D:\Hexo），然后在此文件夹中右键打开Git Bash。执行下面的命令

    hexo init

    等待过后，Hexo会在该目标文件建立所需要文件

  2. 安装node_modules

    npm install

  3. 此时已经可以运行下看看效果了

    hexo server;

    启动服务后，在浏览器中打开http://localhost:4000/，这时可以看到Hexo已为你生成了一篇blog_HelloWorld

    此时的样子是hexo默认的主题，需要更换主题的话，只需要将主题拷贝到theme文件夹中，然后配置_config.yml即可
    具体的配置方法，请参照官网

  4. 创建新的博文

    hexo new "My New Post"

  5. 生成静态网页

    hexo generate

    将博客绑定到GitHub

  • 配置_config.yml文件

    Deployment Docs:http://hexo.io/docs/deployment.html deploy: type:
    更改为
    Deployment Docs:http://hexo.io/docs/deployment.html deploy: type: github repository: git@github.com:xxx/xxx.github.io.git branch: master

  • 开始部署

    $ hexo deploy

    在这里我遇到了个问题，说Deployer not found: github
    解决方法为：

    npm install hexo-deployer-git --save
    然后将配置文件里的github -> git

    再次执行部署命令即可。

    部署完成后再浏览器中http://xxx.github.io/ 查看效果

    每次部署的步骤

    hexo clean //清除缓存 hexo generate //生成静态网页 hexo deploy //部署

    本地调试

  1. 在执行下面的命令后，
    $ hexo g #生成 $ hexo s #启动本地服务，进行文章预览调试</span>

浏览器输入http://localhost:4000，查看搭建效果。此后的每次变更_config.yml 文件或者新建文件都可以先用此命令调试，尤其是当你想调试新添加的主题时。

  1. 可以用简化的一条命令

    hexo s -g
